ELECTRIC GROUP

MODULE: 01_HV

High Voltage

高压组：电车动力与安全的技术中枢,负责整车高压系统（Tractive system）的设计开发、合规验证与功能实现，覆盖电池储能单元、高压配电网络、安全监测模块三大核心板块，是电车动力性能与赛事合规性的技术保障。 （一）电池箱系统开发：储能单元的合规性设计 1.电芯与模组设计 控制模组电压 / 能量 / 重量达标 2.BMS 的使用 实时监测并调控电芯状态，异常快速切断回路 3.箱体结构设计与整体组装 保障结构合规 （二）高压配电网络搭建：动力传输的可靠性实现 1.高压线束设计 2.高压器件选型 （三）安全监测系统落地：赛事合规的核心保障 1.IMD 绝缘监测 2.AMS状态指示 3.安全回路验证 （四）电机及控制器 1.选型适配功率需求，与电池系统参数匹配 2.高强度螺栓固定，支架防振符合赛事要求 3.完成通讯匹配与动力标定，确保输出稳定 新赛季目标 1.设计和组建新的电池箱 2.依据赛规同低压组优化线束布置 3.对现有PCB板进行优化 4.优化后舱布局

MODULE: 02_LV

Low Voltage

UNNC Speeduck 低压组（LV） 是宁波诺丁汉大学诺速鸭车队电气系统的核心技术部门之一，主要负责整车低压电气系统的设计、开发与运行保障，是连接整车各子系统的“神经中枢”。 • 低压系统架构设计：负责整车 12V / 24V 低压供电系统架构规划，包括低压电池、DC-DC 转换、电源分配、接地与冗余设计，保障整车在各工况下的稳定与安全供电。 • 电路板与控制系统开发：参与低压相关电路板的原理图与 PCB 设计，基于 STM32 等 MCU 进行控制板与功能模块开发，完成信号采集、执行控制与系统联调。 • 通信与安全逻辑实现：负责 CAN 总线通信设计与整车信号管理，参与整车安全逻辑的实现与验证，包括急停、使能、故障检测与保护策略，保障车辆符合赛事技术与安全规范。 • 人机交互与功能模块设计：负责方向盘、电控按钮、指示灯、显示屏等人机交互模块的设计与调试，提升驾驶员对车辆状态的感知与操控体验。 • 线束设计与整车集成调试：完成整车低压线束设计、制作与标注，参与赛前系统联调、赛场故障排查与应急维护，确保比赛期间低压系统安全、稳定、可靠运行。

MODULE: 03_SW

Software & Control

软件组负责赋予赛车“智慧”。我们为整车控制器（VCU）编写复杂的底层控制算法，包含扭矩分配请求，扭矩控制系统（TC），上电自检，整车系统控制，并将车身传感器信号进行汇总处理。同时软件组负责展示网站与各类车队工作网页的开发与维护

SPEC: 05_CHASSIS_GEO

Chassis Geometry

底盘组工作围绕设计-仿真-制造-调校展开： 1. 悬架动力学设计与运动学分析：用SolidWorks建悬架模型定硬点，仿真优化几何参数，通过MATLAB、Adams Car/VI-grade处理轮胎数据并做动力学仿真，保障轮胎抓地力。 2. 结构与有限元分析：设计悬架、转向等系统，轻量化迭代零件，用ANSYS等做关键件强度校核。 3. 制造与装配：指导/参与精密加工，完成悬架安装、预埋件检查等底盘组装。 4. 实车测试与调校：依赛道调整底盘参数，采集分析数据，对比仿真与实车表现，优化行驶问题提升圈速。

SPEC: 06_AERODYNAMICS

Aerodynamics Kit

1. 气动构型设计 (Conceptual Design)这是空套组工作的起点，目标是在满足规则的前提下，最大化下压力（Downforce）并最小化风阻（Drag）。翼片设计： 设计前翼（Front Wing）、后翼（Rear Wing）和侧翼的多级翼型组合。底板与扩散器： 利用地面效应（Ground Effect），通过扩散器产生巨大的负压，这是下压力的重要来源。气流引导： 设计侧箱（Sidepods）和定风翼，将气流引导至散热器或避震器，兼顾冷却与减阻。 2. 计算流体力学仿真 (CFD Analysis) 空套组绝大部分的“平时工作”都是在电脑前与流体仿真软件打交道。 建模与网格划分： 使用 ANSYS Fluent 或 Star-CCM+ 对赛车模型进行复杂的网格划分（Mesh），这是仿真精度的基础。 多工况模拟： 模拟赛车在直线加速、转弯（偏航角模拟）以及跟随前车气流时的气动表现。 可视化分析： 通过速度矢量图、压力云图和流线图，寻找气流剥离点，不断迭代优化翼片的攻角和间距。 3. 碳纤维复合材料制造 (Composite Manufacturing) 由于空气动力学套件体积大且要求极轻，空套组通常也是车队的“碳纤维加工中心”。模具制作： 利用木模、泡沫或 3D 打印制作模具。铺层与真空袋： 学习碳纤维布的纹理走向（铺层设计），进行真空抽气，并加入固化剂。结构加强： 内部通常需要夹芯材料（如蜂窝铝或强芯毡）来保证翼片在高速下不发生形变或断裂。 4. 气动性能测试与标定 (Testing & Validation) 仿真再美也需要实测验证。空套组需要确保实验室数据能转化成赛道的秒数，风洞实验： 缩比模型测试。直线滑行测试（Coast-down Test）： 通过传感器测量赛车减速过程，推算阻力系数。流体可视化： 在赛车表面涂抹示踪毛线（Flow Vis），通过毛线动作观察实际气流是否按预期流动。

SPEC: 07_STRUCTURE

Frame Structure

车架是赛车的生命骨架。我们采用高强度钢管桁架结构，通过扭转刚度仿真与疲劳测试，确保了底盘的坚固性。同时，我们对碰撞保护区进行了专项优化，为驾驶员提供最坚实的保护。
